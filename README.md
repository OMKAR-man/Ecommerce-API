# 🛒 E-commerce API

A RESTful API for managing an e-commerce platform.

## 🚀 Features
- User authentication
- Product management
- Cart & orders
- CRUD operations
- Secure endpoints

## 🛠️ Tech Stack
- Node.js / FastAPI
- MongoDB / MySQL
- JWT Authentication

## 📂 API Endpoints

| Method | Endpoint | Description |
|--------|--------|-------------|
| GET | /products | Get all products |
| POST | /products | Add product |
| POST | /login | User login |

## ▶️ How to Run

For Node.js:
npm install  
npm start  

For FastAPI:
uvicorn main:app --reload  

## 📌 Use Cases
- Online shopping platforms
- Backend services
